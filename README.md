# tweetmap
![image](/img/twitterMap.png)
This is a map of geolocation tagged tweets in the US scraped from the Twitter API with a crawler.

The tweet distribution gives us a quick glance at population distribution in the country. Many of the more densely populated areas are along the east and west coasts which can be seen above. The dots in the midwest are sparse however, the mideast seems to have a larger presence. Overall this was a small sample of tweets totalling about 650.
